# 04_DATA_GOVERNANCE

Este directorio está destinado a la gobernanza de datos y el análisis de métricas clave de crecimiento (RevOps) de LidiaLabs.

## Contenido y Alcance

*   **Métricas de Velocidad:** Tiempos de respuesta, tasas de conversión conversacional, embudo de calificación.
*   **Métricas de Negocio:** Costo de Adquisición de Clientes (CAC), Lifetime Value (LTV), y ROI de los pilotos.
*   **Análisis de Cohortes:** Comportamiento de uso y retención de leads calificados agendados en los calendarios.
